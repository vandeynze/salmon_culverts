Research reports for the salmon_culverts repository.

  1. Spending and Activity for Pacific Northwest Salmon Habitat Restoration: A summary of the PNSHP data
  2. Culvert Projects Summary: A summary of where and when culvert projects occur in the PNSHP data
  3. Culvert Cost Pilot Models: A first-cut attempt to model culvert costs using just the PNSHP data
  4. Summary of Spatially Derived Variables for Culvert Projects: A summary of the additional explanatory variables gathered for cost models
  5. Culvert Cost Models with Spatially Explicit Data: An expansion of the early cost models with data gathered from geospatial databases
  6. Culvert Cost Models with Spatial Econometric Methods: Builds on the base cost models with methods from the spatial econometrics literature
  7. Culvert Cost Models with Machine Learning Methods: Compares predictive performance between base OLS method and a suite of regression-tree based machine learning methods
